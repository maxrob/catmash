# Catmash

### Install Project:

```Bash
npm install
```

## API :

### Launch server with hot-reloads

```Bash
npm run server
```

### Launch project linter

```Bash
npm run lint
```

### Build project

```Bash
npm run build
```

### Start project

```Bash
npm run start
```

### Test project

```Bash
npm run test
```
